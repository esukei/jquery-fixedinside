jQuery Fixed-inside demo.

.fixedinside()

親コンテナの中でのみ固定配置になります。

The element is fixed inside the parent container.

画面についてくるサイドメニューなどに便利です。

It's useful to make following side menu.

表示される高さが固定要素の高さに足りない場合は固定しません。ブラウザの高さを縮めてお試しください。

If viewport height is not enough showing the fixed element, fixed-inside will be disabled. Change browser height to try.

親コンテナの上下paddingが考慮されます。

The parent container's padding affects to the fixed element.
